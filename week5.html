Неделя 5 - Старший класс

День 1. Атрибуты тега input

При помощи дополнительнх атриботов

  • required - делает поле обязательным для заполнения
  • placeholder - текст в поле который изчезает при фокусе
  • autocomplete - автозаполнение поля
  • pattern - задает формат ввода данных для поля.
  • min - задает минимальное значение
  • max - задает максимальное значение
  • multiple - разрешает указать несколько почтовых адресов (через запятую) или файлов для загрузки
  • step - задет шаг и актуален для полей числовых и ползунка
  • autofocus - задает автоматический фокус для указанного поля
  • form - привязывает поле к определенной форме
  • list - используется для привязки к тегу <datalist> чтобы указать на список вариантов

День 2. Человекоформа

<label> - этот тег связывает текст и поле формы. То есть, при клике на тексте, активируется поле.

Связыватся можно двумя способами. Перый - разместить все внутри тега mytag <label>. Второй при помощи специального атрибута и идентификатора.

<select> - задает список в котором можно выбрать определенные значения. Значения толи (пункты списка) задаются при помощи тега <option>

<optgroup> - объединяет пункты списков в отдельные группы

<output> - через этот тег можно вывести результат поля формы или его дополнительно обработать при выводе.

<datalist> - создает список вариантов для заполнения поля. Активируется данный список при фокусе на поле (двойной щелчок) либо про вводе информации.

<textarea> - поле для ввода большого текста. Текст виден в несколько строк

День 5. Семантические элементы HTML5.

<main> - внем должно содержаться главный контент страницы.

1. Этот тег должен быть уникальным и встречаться один раз на странице

2. Он не должен быть внутри других структурных тегов

Если современные браузеры не будут понимать как отображать данный тег, пропишите ему следующие стили:

main {display:block;}

<mark> - создает на странице выделенный текст. В некоторых браузерах он выделяется по умолчанию желтым.

Это выделенный текст. Используйте его для важных заметок.

<time> - в нем указываем дату или время или то и другое.

Атрибуты:

  • datetime - дата и время в тексте.
  • pubdate - указывает дату публикации.

1980.01.11 Мой день, а может и не мой

<figure> - в нем мы можем группировать какие-либо элементы сайта

<figcaption> - в нем размещаеться элементы группированых элементов сайта.

Этот тег должен быть внутри тега <figure> И он должен быть либо самым первым либо самым последним
